# 開発ログ

## これまでに完成している内容

### 基本機能

- React、TypeScript、Viteを使ったアプリの土台
- IndexedDBへの日付別メモ保存と読み込み
- メモの新規作成、一覧表示、本文・タグの編集、削除
- 今日への移動、日付選択、前日・翌日への移動
- メモ件数と絞り込み後の件数表示

### 整理・検索

- 6種類のタグ付け
- 複数タグによる日付内の絞り込み（いずれかのタグに一致）
- 全期間のメモを対象にした検索専用画面
- 本文、タグ、日付文字列を対象にした部分一致検索
- 検索結果から該当日へ移動し、対象メモを一時的に強調表示
- 検索語がない場合の最新20件表示

### 画像

- 画像1枚の添付、送信前プレビュー、取り消し、保存後の表示
- 保存前の画像縮小とJPEG変換による容量削減
- 画像だけのメモ作成

### データ保全

- メモと画像を含むJSONバックアップの書き出し
- JSONバックアップの読み込み
- 過去データに画像やタグがない場合の補完処理

### UI・配布

- PC向けサイドバーとスマートフォン向け三本線メニュー
- 画面幅760px以下、420px以下を考慮したレスポンシブスタイル
- PWA用Manifest、アプリアイコン、Service Worker
- 本番環境でのService Worker登録
- GitHub Actionsによる `main` ブランチからGitHub Pagesへの自動公開設定
- READMEへのローカル実行、公開、バックアップ手順の記載
